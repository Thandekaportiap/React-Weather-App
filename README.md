<h1>React-Weather-App</h1>

## Run Locally
Clone the project
```bash
  git clone https://github.com/Thandekaportiap/React-Weather-App.git
```
Go to the project directory
```bash
  cd React-Weather-App
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run start
```
